# Technologies Used

1. Language : Java 11
2. Framework : Spring Boot
3. Database : Mongodb
4. Maven
5. Swagger for documentation

# Build/Execute

Import this maven project in your IDE (I am using IntelliJ IDEA).
Please run docker/local-dev.yml -> services to set up mongo db
